Nome do componente curricular: Modelagem de Dados
Pré-requisitos: não há
Carga horária total: 72 hs
Carga horária p/ prática: 36h

Carga horária p/ teoria: 36h

Objetivos gerais:
O objetivo desse curso é oferecer aos alunos de graduação uma abordagem teórica e prática
dos principais métodos para análise quantitativa em negócios. A abordagem segue o uso de
computadores para ajudar no processo de tomada de decisão utilizando ferramentas e
métodos científicos.
Objetivos específicos:
1. Apresentar as principais técnicas para modelagem computacional de problemas para
tomada de decisões.
Ementa: 1. Introdução a Otimização e Programação Linear, 2. Análise de Sensibilidade, 3.
Modelagem de Rede, 4. Programação Inteira, 5. Análise de Regressão Simples, 6. Análise de
Regressão Múltipla, 7. Previsão de Séries Temporais, 8. Tópicos Específicos em Análise de
Dados.
